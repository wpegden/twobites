% Paper-defined macros for the viewer's KaTeX renderer.

% The viewer fetches this file at init (via /api/paper-header.tex) and
% feeds the \newcommand / \renewcommand bodies to KaTeX as macros.
% Mirrors the paper preamble at paper/reference.tex (lines ~21-75)
% with the following adjustments:
%   - drop the \todo-using author markup macros (\Flo, \Paul, \Dylan,

%   - drop \JN, \SS — author-color markup, harmless to omit
%   - add \PP → \mathbb{P}; the paper uses \P / \Pr but agents
%     occasionally emit \PP by analogy. Make it work rather than fail.
%
% Format: one \newcommand or \renewcommand per line, body on the same
% line. The viewer parses with a simple regex; nested braces inside a
% body are fine but don't put two commands on one line.
