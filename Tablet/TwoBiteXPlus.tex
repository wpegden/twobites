\begin{definition}
For a two-bite sample \(\omega\), a finite vertex set \(I\subseteq[n]\), and a
base vertex \(x\in V_R\cup V_B\),
\[
\noderef{TwoBiteXPlus}(\omega,I,x)
=\{v\in I: v\in\noderef{TwoBiteLiftedPositiveNeighborhood}(\omega,x)\}.
\]
Thus \(X_x^+(I)\) is exactly \(I\) filtered by the one-sided lifted
neighborhood of \(x\).  This is the paper's one-sided version of \(X_x(I)\)
used for same-color deletions: a same-color triangle is charged to the
lexicographically latest base edge, so only neighbors with larger base index
than \(x\) contribute to \(X_x^+(I)\).
\end{definition}
