\begin{helper}
Let \(n,m\) be natural numbers, let \(U_0\subseteq \operatorname{Fin} n\),
let \(\rho:\operatorname{Fin}n\to\operatorname{Fin}m\), and let
\(\eta:U_0\to\operatorname{Fin}m\).  Put \(R_i=\{v:\rho(v)=i\}\).  Let
\(X_{\mathrm{all},\eta}\) be the set of injections
\(\phi:\operatorname{Fin}n\hookrightarrow\operatorname{Fin}m\times
\operatorname{Fin}m\) whose first coordinate is \(\rho\) and whose second
coordinate on \(U_0\) is \(\eta\).  For each row \(i\), let \(d_i\) be the
number of injective maps \(f:R_i\to\operatorname{Fin}m\) satisfying
\(f(u)=\eta(u)\) for every \(u\in U_0\cap R_i\).  Then
\[
|X_{\mathrm{all},\eta}|=\prod_i d_i.
\]
\end{helper}
\begin{proof}
Map a global injection \(\phi\in X_{\mathrm{all},\eta}\) to the tuple of
row maps \(f_i:R_i\to\operatorname{Fin}m\) defined by
\(f_i(v)=(\phi(v))_2\).  If \(v,w\in R_i\) and \(f_i(v)=f_i(w)\), then
\(\phi(v)\) and \(\phi(w)\) have the same first coordinate \(i\) and the
same second coordinate, so injectivity of \(\phi\) gives \(v=w\).  The eta
condition on \(\phi\) gives \(f_i(u)=\eta(u)\) on \(U_0\cap R_i\), so each
\(f_i\) is counted by \(d_i\).

Conversely, from a tuple of row maps counted by the \(d_i\), define
\(\phi(v)=(\rho(v),f_{\rho(v)}(v))\).  Equality \(\phi(v)=\phi(w)\) first
forces \(\rho(v)=\rho(w)\), so \(v\) and \(w\) lie in the same row; rowwise
injectivity then gives \(v=w\).  The first coordinate is \(\rho\), and the
row eta condition gives the required second coordinate on \(U_0\).  These
two constructions are inverse, so \(X_{\mathrm{all},\eta}\) is in bijection
with the finite cartesian product of the row sets counted by \(d_i\).
Taking cardinalities gives the displayed product identity.
\end{proof}
