\begin{definition}
For a sample \(\omega\), the value of a tagged base-edge coordinate is the
corresponding red or blue adjacency value in the sampled base graph.  Thus a
coordinate \(\operatorname{red}(a,b)\) has value
\((G_R(\omega)).\operatorname{Adj}(a,b)\), while a coordinate
\(\operatorname{blue}(a,b)\) has value
\((G_B(\omega)).\operatorname{Adj}(a,b)\).
\end{definition}
