\begin{definition}
For a class \(\mathcal A\) of base vertices,
\(\noderef{TwoBiteProjectedPairsClosedInClass}(\omega,I,\mathcal A)\) is the
set of red/blue tagged projected pairs in
\(\noderef{TwoBiteProjectionPairSet}(\omega,I)\) that are realized inside
\(X_x(I)\) for at least one witness \(x\in\mathcal A\), using the red
projection for red-tagged pairs and the blue projection for blue-tagged pairs.
\end{definition}
