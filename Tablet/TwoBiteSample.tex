\begin{definition}
A two-bite sample with fixed parameters \(n,m,p\) is the random object used in
the paper's construction.  It consists of vertex sets
\[
V_R=\{r_1,\ldots,r_m\},\qquad
V_B=\{b_1,\ldots,b_m\},\qquad
V(G)=\{v_1,\ldots,v_n\},
\]
two base graphs
\[
G_R\quad\text{on }V_R,
\qquad
G_B\quad\text{on }V_B,
\]
and an injective map
\[
\pi:V(G)\hookrightarrow V_R\times V_B.
\]
For each final vertex \(v\), write
\(\pi(v)=(\pi_R(v),\pi_B(v))\).  For a set \(X\subseteq V(G)\), the images
\(\pi(X)\), \(\pi_R(X)\), and \(\pi_B(X)\) are the corresponding product, red,
and blue projections.
\end{definition}
