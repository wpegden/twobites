\begin{definition}
For a base size \(m\), the finite set
\(\noderef{TwoBiteTerminalCoordinateUniverse}(m)\) is the set of all tagged
red and blue base-edge coordinates
\[
\operatorname{red}(a,b),\qquad \operatorname{blue}(a,b)
\]
with \(a,b\in[m]\) and \(a<b\).  Thus each unordered non-loop base edge is
represented once, by its increasing endpoint order, in each color copy.  This
is the coordinate universe from which terminal Bernoulli factors are exposed
after the pre-terminal history has been fixed.
\end{definition}
